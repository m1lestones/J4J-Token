## About https://j4-j-token.vercel.app 

Justice for Joey ($J4J) is a meme coin created to raise awareness about government fraud and accountability. Inspired by Nick Shirley's exposé of the Minnesota daycare fraud scandal that allegedly involved over $9 billion in fraudulent claims.

### The Story

Joey is the fictional child Nick Shirley used to expose Minnesota's billion-dollar daycare fraud scandal. Through his investigation, Nick uncovered:

- 🏚️ **Empty Daycares** - Licensed for 99 children, receiving millions in taxpayer funds
- 📝 **"Quality Learing Center"** - A daycare that can't spell "Learning" received $4M in state funds  
- 💰 **$9 Billion Missing** - Federal prosecutors estimate $9B in fraud
- 🔍 **One Man's Investigation** - 23-year-old Nick Shirley's 100M+ view exposé led to FBI involvement

## Getting Started

1. Clone this repository
```bash
git clone https://github.com/yourusername/j4j-token.git
cd j4j-token
```

2. Open `index.html` in your browser or serve with a local server
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

3. Visit `http://localhost:8000` to view the site
